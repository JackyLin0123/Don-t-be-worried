# Don't Be Worried

A small Android social-comfort experiment inspired by the communication anxiety many young people feel in modern messaging apps.

When a message is not answered immediately, people often begin to overthink: *Did I say something wrong? Are they ignoring me? Should I send another message?* This project explores a simple emotional-design idea: placing a familiar "typing..." indicator over a chat screen so the waiting moment feels less empty, less uncertain, and less stressful.

The app displays a floating typing indicator bubble on top of other apps. It does not read messages, modify chat apps, or detect whether another person is actually typing. It is a lightweight, user-controlled visual overlay for
