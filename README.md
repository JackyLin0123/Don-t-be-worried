# Don't Be Worried

An Android Kotlin project inspired by the social anxiety young people often feel while waiting for message replies.

When someone does not reply immediately, the empty waiting time can make people overthink. This app adds a floating "typing..." bubble over the screen to create a softer sense of presence and reduce that anxious feeling.

## Features

- Floating typing indicator overlay
- Draggable bubble
- Animated three-dot typing effect
